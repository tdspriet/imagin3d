\section{Baseline Models}

Because there are currently no direct equivalents to the Imagin3D framework that accept heterogeneous, spatially-arranged moodboards, selecting an appropriate baseline requires isolating the specific contributions of the orchestration pipeline. The current state-of-the-art in direct 3D generation is heavily constrained by strict input limitations. For instance, Trellis v1 processes either a single text prompt or a single image exclusively, while Trellis v2 is entirely restricted to single-image conditioning. % should mention that there are TrellisV2 or Hunyuan3D-2 variants that accept text online but all go through an image(s) bottleneck, so they all have a cascaded design

\section{Measuring Performance}

Evaluating the effectiveness of moodboard-based conditioning presents unique challenges. Understanding spatial relationships and volumetric properties requires mental rotation and multi-perspective evaluation that 2D images do not. Consequently, we employ a hybrid evaluation strategy consisting of subjective user studies and objective distance metrics to assess output quality and alignment with user intent.

\subsection{Subjective Metrics}

Because evaluating multi-faceted stylistic guidance is inherently qualitative, human evaluation remains the most critical performance indicator. We conduct user studies focusing on two distinct evaluation paradigms:

\begin{itemize}
    \item \textbf{A/B Preference Testing:} Participants are presented with a source moodboard alongside anonymized 3D models generated by Imagin3D and the defined baseline. Users must blindly select which model best harmonizes the disparate influences and overall intent of the moodboard.
    \item \textbf{Targeted Recreation Task:} To evaluate the system's utility in professional pipelines, participants are given a target visual concept and asked to generate it using both the Baseline manual workflow and the Imagin3D moodboard interface. We evaluate this task across two dimensions:
    \begin{itemize}
        \item \textit{Time-to-Completion:} The total time required to achieve a satisfactory result, measuring whether the moodboard reduces the traditional ``generate-and-hope'' loop.
        \item \textit{Intuitiveness:} A 5-point Likert scale rating how naturally the user could express their structural and stylistic goals.
    \end{itemize}
\end{itemize}
